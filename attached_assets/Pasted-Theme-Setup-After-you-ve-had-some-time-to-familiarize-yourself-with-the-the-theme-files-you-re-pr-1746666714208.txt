## üîç Global SEO Settings

The foundation of your site's visibility starts in `src/config/settings.js`:

```javascript
export default {
  title: `Dr. Jan Duffy | Premier Luxury Real Estate - Consenza Estates Las Vegas`,
  description: `Experience unparalleled luxury with Dr. Jan Duffy, your elite real estate specialist for prestigious properties in Las Vegas.`,
  url: `https://consenzaestates.com`, // No trailing slash!
  name: `Dr. Jan Duffy Luxury Real Estate`, // Used for copyright and other branding elements
  enableThemeSwitcher: true,
};
```

These settings serve as global defaults when page-specific values aren't provided.

## üé® Logo Customization

Your brand identity is central to client recognition. Update your logo in `src/components/Logo.astro`:

**Before:**
```astro
<p class="odyssey-logo">Odyssey</p>
```

**After:**
```astro
<!-- Option 1: Text-based logo with luxury styling -->
<p class="luxury-logo">Dr. Jan Duffy</p>

<!-- OR -->

<!-- Option 2: Image-based logo -->
<img src="/assets/images/logo.svg" alt="Dr. Jan Duffy Luxury Real Estate" class="luxury-logo-img" />
```

**Pro Tip:** For SVG logos, paste the SVG code directly for best performance and styling options.

## üåê Favicon & Social Images

### Favicon
Replace `public/favicon.png` with your custom logo in PNG format.

For advanced favicon options, update the following in `src/components/head/BaseHead.astro`:
```html

```

## üó∫Ô∏è Sitemap Configuration

For optimal SEO, configure your sitemap in `astro.config.mjs`:

```javascript
import { defineConfig } from 'astro/config';
import lit from '@astrojs/lit';
import sitemap from "@astrojs/sitemap";

export default defineConfig({
  site: 'https://consenzaestates.com/',
  // Generates sitemap.xml automatically
  sitemap: true,
  integrations: [lit(), sitemap()]
});
```
